# 🛡️ Detector de Elementos PPE (Equipo de Protección Personal)

Desarrollado por **Ralph Castellanos Couott**

## 📋 Descripción
Aplicación web para detección de Elementos de Protección Personal (PPE) usando YOLOv8.

## 🎯 Elementos detectables
- Casco
- Chaleco
- Guantes  
- Gafas
- Mascarilla
- Calzado de seguridad

## 🚀 Características
- ✅ Subida de imágenes desde dispositivo
- ✅ Análisis por URL
- ✅ Tabla con porcentajes de confianza
- ✅ Filtro de detecciones < 50%
